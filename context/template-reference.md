# Референс шаблона 159.194.215.53

## Стек (по следам production-бандла)

- Vite + React 18
- Wouter (маршрутизация)
- Zustand (состояние)
- Radix UI / shadcn-подобные компоненты
- Lucide icons
- Open Sans + Caveat (Google Fonts)
- Replit dev banner (в dev)

## UX-поток «Насосные установки»

| Шаг | Заголовок | Элементы |
|-----|-----------|----------|
| 1 | Подбор насосного оборудования Стрела | Карточки классов продукции |
| 2 | Линейка насосных установок | Карточка BPS-W (и др.) |
| 3 | Тип установки | Хоз-пит / ПНС |
| 4 | Подбор насосной установки BPS-W | Форма параметров, подбор, опции, результат |

## Поля формы (хоз-пит, BPS-W)

- Тип насоса (combobox)
- Рабочие / резервные насосы (number)
- Производительность м³/ч, напор, статический, гарантированный (number)
- Тип жидкости, температура
- Кнопки: Сбросить, Подобрать, Сформировать станцию
- Опции: PN, Управление, Фильтр, Подключение, чекбоксы (клапан, виброопоры, изоляция, бак)
- Блоки: подходящие насосы, результат конфигурации

## Визуальный стиль

- Синяя боковая панель с вертикальным логотипом «СТРЕЛА»
- Карточки: белый верх (изображение), синий низ (заголовок + описание)
- Кнопка «Войти» в шапке

## Соответствие в нашем проекте

- Поток описан в `config/profiles/default/wizard/`
- Тема по умолчанию: `packages/theme-strela`
- Скриншоты/детали UI — сверять с живым шаблоном по IP выше
