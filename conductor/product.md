# PumpStation Base — Product

## Описание

Веб-приложение для подбора насосных станций и смежного оборудования: многошаговый сбор параметров, расчёт на сервере, формирование конфигурации и PDF.

## Пользователи

- Инженеры-проектировщики — основной сценарий подбора
- Менеджеры OEM — разные брендинги и PDF под заказчика
- Внутренние редакторы (без программирования) — тексты, шаги визарда, профили клиентов через Cursor

## Ключевые возможности

1. Многоэтапный визард (класс → линейка → тип → форма)
2. Подбор насосов и сборка станции (алгоритм на backend)
3. Профили: тема + алгоритм + БД + шаблон PDF
4. Учёт пользователей и настроек (этап 2)

## Успех

- Новый клиент подключается через `config/profiles/` без форка кода
- Редакторы меняют шаги и тексты без правок TypeScript/Python
